## Getting Started
These are the steps to building this app yourself:<br>

1. poetry init -n
1. poetry add Django
1. poetry shell
1. tree (keep an eye on what you are doing)
1. python manage.py runserver
1. code .
1. add .gitignore 
1. python manage.py migrate

Always keep your eyes on:
- views
- urls
- template

Go to settings.py
DEBUG = True
